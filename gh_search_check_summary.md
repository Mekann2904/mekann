作成した `gh_search.sh` は正常に動作しています。

1. **ファイル内容の読み込み**:

### ツール仕様まとめ

- **パス**: `.pi/tools/gh_search.sh`
- **機能**: `gh api search` のラッパー
- **使用方法**:
  ```bash
  .pi/tools/gh_search.sh <クエリ> [オプション]
  ```
- **オプション**:
  - `-t, --type`: 検索タイプ (`code` [デフォルト], `issues`, `commits`, `repositories`)
  - `-l, --limit`: 取得件数 (デフォルト: 5)
  - `-r, --repo`: 対象リポジトリ (省略時はカレントリポジトリ)
  - `-e, --extension`: 拡張子フィルタ (`code` 検索時のみ)

このツールを使用して、GitHub上のリポジトリ情報を効率的に検索できます。
