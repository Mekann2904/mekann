# Support

Mekann は実験的な Pi extension suite です。現時点では best-effort support です。

## Where to ask

- Bug report: GitHub issue を作成してください。
- Feature request: 背景、期待する workflow、代替案を添えて GitHub issue を作成してください。
- Security issue: [SECURITY.md](./SECURITY.md) に従って非公開で報告してください。
- Usage question: まず [README](./README.md)、[docs/README.md](./docs/README.md)、各 feature README を確認してください。

## Useful details for issues

- OS と terminal emulator。
- Node.js version。
- Pi version。
- Mekann commit hash。
- 実行した command と結果。
- 関連する `mekann.json` 設定。ただし token、secret、private URL は除いてください。
