## 7つの主命題

### 命題1: 世界論 (Die Welt)

```
1. Die Welt ist alles, was der Fall ist.
   世界は、事実にあるものすべてである。

1.1 Die Welt ist die Gesamtheit der Tatsachen, nicht der Dinge.
    世界は物の総体ではなく、事実の総体である。
```

**核心**: 世界 = 事実の総体（物ではない）

**分析ポイント**:
- 存在論的転換：物から事実へ
- 事実の独立性（1.21）
- 論理空間内の事実（1.13）

---

### 命題2: 対象論 (Die Gegenstände)

```
2. Was der Fall ist, die Tatsache, ist das Bestehen von Sachverhalten.
   事実であるものは事態の成立である。

2.01 Der Sachverhalt ist eine Verbindung von Gegenständen.
    事態は対象の結合である。

2.02 Der Gegenstand ist einfach.
    対象は単純である。
```

**核心**: 事実 = 事態の成立、事態 = 対象の結合

**階層構造**:
```
世界 (Welt)
    │
    └── 事実 (Tatsache)
            │
            └── 事態 (Sachverhalt)
                    │
                    └── 対象 (Gegenstand)
```

**分析ポイント**:
- 対象の単純性（分析不可能性）
- 対象の可能性包含（2.014）
- 世界の実体としての対象（2.021）

---

### 命題3: 思想論 (Der Gedanke)

```
3. Das logische Bild der Tatsachen ist der Gedanke.
   事実の論理的な像は思想である。
```

**核心**: 思想 = 事実の論理的像

**分析ポイント**:
- 思考可能 = 可能（3.02）
- 非論理的思考の不可能性（3.03）
- 言語による思想の表現（3.1）

---

### 命題4: 命題論 (Der Satz)

```
4. Der Gedanke ist der sinnvolle Satz.
   思想は意味ある命題である。

4.01 Der Satz ist ein Bild der Wirklichkeit.
    命題は実在の像である。
```

**核心**: 命題 = 思想の言語的表現 = 世界の像

**像の理論 (Bildtheorie)**:
```
像 (Bild)
    │
    ├── 要素 (Elemente) → 対象に対応
    ├── 構造 (Struktur) → 要素の関係
    └── 描写形式 (Form der Abbildung) → 論理的形式
```

**分析ポイント**:
- 命題の意味（4.022）
- 真理条件（4.431）
- トートロジーと矛盾（4.46）

---

### 命題5: 真理関数論 (Die Wahrheitsfunktion)

```
5. Der Satz ist eine Wahrheitsfunktion der Elementarsätze.
   命題は要素命題の真理関数である。
```

**核心**: すべての命題は要素命題の真理関数として分析可能

**真理表（例: 2つの要素命題）**:
```
p │ q │ p∧q │ p∨q │ p→q │ ¬p
──┼───┼─────┼─────┼─────┼────
T │ T │  T  │  T  │  T  │ F
F │ T │  F  │  T  │  T  │ T
T │ F │  F  │  T  │  F  │ F
F │ F │  F  │  F  │  T  │ T
```

**分析ポイント**:
- N演算（5.5）
- 推論の理論（5.11-5.1363）
- 因果性の否定（5.136）

---

### 命題6: 論理・数学論 (Logik und Mathematik)

```
6. Die allgemeine Form der Wahrheitsfunktion ist: [p, ξ, N(ξ)].
   真理関数の一般形式は [p, ξ, N(ξ)] である。

6.1 Die Sätze der Logik sind Tautologien.
    論理の命題はトートロジーである。

6.2 Die Mathematik ist eine logische Methode.
    数学は論理的な方法である。
```

**核心**: 論理 = トートロジー、数学 = 論理的方法

**分析ポイント**:
- 論理の超越論性（6.13）
- 数学の性質（6.2-6.241）
- 自然法則と因果性（6.3-6.374）
- 倫理の超越論性（6.4-6.5）

---

### 命題7: 限界論 (Die Grenze)

```
7. Wovon man nicht sprechen kann, darüber muss man schweigen.
   語り得ぬものについては、沈黙せねばならない。
```

**核心**: 言語の限界を認識し、語り得ぬものには沈黙する

**梯子の比喩（6.54）**:
> 私の命題を理解する者は、最終的にそれらを無意味として認識する。
> 彼がそれらを通り抜け、それらの上に立って、それらを超えたとき。
> （彼はいわば梯子を登った後、それを投げ捨てなければならない。）

| 概念 | ドイツ語 | 定義 | 関連命題 |

| **世界** | Welt | 事実の総体 | 1.1 |
| **事実** | Tatsache | 事態の成立 | 2 |
| **事態** | Sachverhalt | 対象の結合 | 2.01 |
| **対象** | Gegenstand | 単純な要素 | 2.02 |
| **像** | Bild | 事実のモデル | 2.12 |
| **思想** | Gedanke | 事実の論理的像 | 3 |
| **命題** | Satz | 思想の言語的表現 | 3.1, 4 |
| **名前** | Name | 対象を指す単純記号 | 3.202 |
| **意味** | Sinn | 像が表現するもの | 2.221 |
| **意義** | Bedeutung | 名前が指す対象 | 3.203 |
| **要素命題** | Elementarsatz | 名前の直接の結合 | 4.21 |
| **真理関数** | Wahrheitsfunktion | 要素命題の関数 | 5 |
| **トートロジー** | Tautologie | すべての真理可能性で真 | 4.46 |

### 概念階層

```
                        世界 (Welt)
                           │
           ┌───────────────┼───────────────┐
           │               │               │
         事実          像 (Bild)        言語
      (Tatsache)          │           (Sprache)
           │               │               │
         事態           要素            命題
     (Sachverhalt)     (Element)        (Satz)
           │               │               │
         対象          対象への          名前
     (Gegenstand)      指示             (Name)
```

### 概念間関係

| 関係タイプ | 説明 | 例 |
|-----------|------|-----|
| **包含** | A ⊃ B | 事実 ⊃ 事態 |

| **表現** | A → B | 命題 → 思想 |

## 主要論証

### 論証1: 世界の実体の必要性

```
前提1 (2.021): 対象は世界の実体を形成する
前提2 (2.0211): 実体がなければ、命題の意味は他の命題の真理に依存する
前提3 (2.0212): 実体がなければ、世界の像を描くことが不可能

結論: 世界には実体（分析不可能な対象）が存在しなければならない

タイプ: 演繹（条件付き証明）
妥当性: ○ 形式的に有効
反論: 対象の「単純性」が論証されていない
```

### 論証2: 命題の像的性質

```
前提1 (2.12): 像は実在のモデルである
前提2 (2.141): 像は事実である
前提3 (2.18): 像と被像物は論理的形式を共有する
前提4 (3.14): 命題記号は事実である

結論 (4.01): 命題は実在の像である

タイプ: 類推
妥当性: △ 類推に基づく、直観的に妥当
反論: すべての命題が像的であるかは疑問
```

### 論証3: 倫理は語り得ない

```
前提1 (6.41): 世界の意味は世界の外側にある
前提2 (6.42): 命題はより高次のものを表現できない
前提3 (6.421): 倫理は超越論的である

結論 (7): 倫理について語ることはできない

タイプ: 演繹
妥当性: ○ 形式的に有効（前提を受け入れれば）
反論: 「超越論的」の概念が循環的
```

### 論証4: 哲学の性質

```
前提1 (4.003): 哲学的命題の多くは無意味
前提2 (4.112): 哲学の目的は思想の論理的明確化
前提3 (6.53): 正しい方法は無意味を指摘すること

結論 (4.112): 哲学は活動であり、命題の体系ではない

タイプ: 演繹
妥当性: ○ 自己適用的
反論: Tractatus自体が哲学的命題の体系
```

## 「言える/示せる/語り得ぬ」の区分

### 三領域の定義

```
┌─────────────────────────────────────────────────────────────┐
│                      言えること                             │
│                     (Sagbares)                              │
│                                                             │
│  • 事実についての命題                                        │
│  • 自然科学の命題                                            │
│  • 真または偽として検証可能                                  │
│  • 要素命題とその真理関数                                    │
│                                                             │
│  特徴: 意味(Sinn)を持つ                                      │
│                                                             │
│  例: 「雪は白い」「aRb」「2+2=4」                            │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                     示せること                              │
│                    (Zeigbares)                              │
│                                                             │
│  • 論理的形式                                                │
│  • 言語の構造                                                │
│  • 命題の意味の形式                                          │
│  • 対象の論理的性質                                          │
│                                                             │
│  特徴: 「言おうとすると」無意味になる                        │
│        命題そのものが「示す」                                │
│                                                             │
│  例: 「pと~pが矛盾する」とは言えないが、                    │
│       ~(p ∧ ~p)がトートロジーであることが示される           │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                    語り得ぬこと                             │
│                   (Unsagbares)                              │
│                                                             │
│  • 倫理（善悪、価値）                                        │
│  • 美学（美醜）                                              │
│  • 神秘（世界の「あること」）                                │
│  • 宗教的・形而上学的真理                                    │
│                                                             │
│  特徴: 命題で表現不可能                                      │
│        「示される」こともできない                            │
│                                                             │
│  対処: 沈黙                                                  │
│                                                             │
│  例: 「世界の意味」「善とは何か」                            │
└─────────────────────────────────────────────────────────────┘
```

### 「示す」と「言う」の区別 (4.121-4.1212)

```
> Der Satz kann die logische Form nicht darstellen, 
> sie spiegelt sich in ihm. (4.121)
> 命題は論理的形式を表現できない、それは命題の中に映る。

> Was gezeigt werden kann, kann nicht gesagt werden. (4.1212)
> 示されることは言われることができない。
```

**実例**:
- 「『fa』は対象aについて語っている」とは**言えない**
- しかし、『fa』という命題の形式がaを**示している**
- 同様に、「pとqが矛盾する」とは**言えない**
- しかし、~(p ∧ q)がトートロジーであることがそれを**示している**

## 哲学的意義

### Tractatusの核心的主張

1. **世界 = 事実の総体**
2. **言語 = 世界の論理的像**
3. **命題の意味 = 真理条件**
4. **論理 = トートロジーの体系**
5. **哲学 = 言語批判**
6. **倫理・価値 = 語り得ぬ**

### 哲学史における位置

| 側面 | Tractatus以前 | Tractatus | Tractatus以降 |
|------|--------------|-----------|---------------|
| **世界** | 物・実体 | 事実の総体 | 構成的世界観 |
| **言語** | 思考の表現 | 世界の像 | 使用の理論 |
| **哲学** | 体系構築 | 言語批判 | 分析哲学 |
| **論理** | 思考法則 | トートロジー | 形式意味論 |

### 後期ヴィトゲンシュタインとの関係

Tractatus → 『哲学的探求』への転換：

| 側面 | Tractatus | 哲学的探求 |
|------|-----------|------------|
| **言語** | 世界の像 | 言語ゲーム |
| **意味** | 真理条件 | 使用 |
| **規則** | 適用は確定的 | 適用は創造的 |
| **哲学** | 沈黙 | 治療的活動 |

## 分析への応用

Tractatusを`logical-analysis`スキルで分析する場合のポイント：

### 構造分析

- 番号体系（1, 1.1, 1.11, ...）は依存関係を示す
- 上位命題から下位命題への展開
- 命題7が全体の結論

### 概念分析

- 7つの主命題が主要概念を導入
- 概念の階層性（世界→事実→事態→対象）
- 「像」「形式」「構造」の相互定義

### 論証評価

- 暗黙の前提が多い
- 類推的推論が中心（像の理論）
- 自己言及的パラドックス（6.54）

### 統合的解釈

- 核心的主張：言語の限界
- 強み：論理的厳密さ、体系性
- 限界：前提の自明性、循環性
