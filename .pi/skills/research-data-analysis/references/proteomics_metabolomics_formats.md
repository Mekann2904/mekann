# プロテオミクス・メタボロミクスファイル形式リファレンス

本リファレンスは、プロテオミクス、メタボロミクス、脂質omics、および関連するオミクスワークフローに特有のファイル形式をカバーしています。

## 質量分析ベースのプロテオミクス

### .mzML - Mass Spectrometry Markup Language
**説明:** MSデータの標準XML形式
**典型的なデータ:** MS1およびMS2スペクトル、保持時間、強度
**使用例:** プロテオミクス、メタボロミクスパイプライン
**Pythonライブラリ:**
- `pymzml`: `pymzml.run.Reader('file.mzML')`
- `pyteomics.mzml`: `pyteomics.mzml.read('file.mzML')`
- `pyopenms`: OpenMS Pythonバインディング
**EDAアプローチ:**
- スキャン数およびMSレベル分布
- トータルイオンクロマトグラム（TIC）解析
- ベースピーククロマトグラム（BPC）
- m/zカバレッジと分解能
- 保持時間範囲
- プレカーサー選択パターン
- データ完全性
- 品質管理メトリクス（ロックマス、標準物質）

### .mzXML - Legacy MS XML Format
**説明:** 旧式のXMLベースMS形式
**典型的なデータ:** メタデータ付き質量スペクトル
**使用例:** レガシープロテオミクスデータ
**Pythonライブラリ:**
- `pyteomics.mzxml`
- `pymzml`: mzXML読み込み可能
**EDAアプローチ:**
- mzMLと同様
- 形式バージョン互換性
- 変換品質検証
- メタデータ保持確認

### .mzIdentML - Peptide Identification Format
**説明:** ペプチド同定のためのPSI標準形式
**典型的なデータ:** ペプチド・スペクトルマッチ、タンパク質、スコア
**使用例:** 検索エンジン結果、プロテオミクスワークフロー
**Pythonライブラリ:**
- `pyteomics.mzid`
- `pyopenms`: MzIdentMLサポート
**EDAアプローチ:**
- PSM数およびスコア分布
- FDR計算とフィルタリング
- 修飾解析
- ミスクレアージェ統計
- タンパク質推論結果
- 検索パラメータ検証
- デコイヒット解析
- ランク1 vs 下位ランク

### .pepXML - Trans-Proteomic Pipeline Peptide XML
**説明:** ペプチド同定用TPP形式
**典型的なデータ:** 統計的検証付き検索結果
**使用例:** プロテオミクスデータベース検索出力
**Pythonライブラリ:**
- `pyteomics.pepxml`
**EDAアプローチ:**
- 検索エンジン比較
- スコア分布（XCorr、期待値等）
- チャージ状態解析
- 修飾頻度
- PeptideProphet確率
- タンパク質カバレッジ
- スペクトルカウンティング

### .protXML - Protein Inference Results
**説明:** TPPタンパク質レベル同定
**典型的なデータ:** タンパク質グループ、確率、ペプチド
**使用例:** タンパク質レベル解析
**Pythonライブラリ:**
- `pyteomics.protxml`
**EDAアプローチ:**
- タンパク質グループ統計
- 倹約的タンパク質セット
- ProteinProphet確率
- タンパク質あたりのカバレッジとペプチド数
- ユニーク vs 共有ペプチド
- タンパク質分子量分布
- GOタームエンリッチメント準備

### .pride.xml - PRIDE XML Format
**説明:** Proteomics Identifications Database形式
**典型的なデータ:** 完全なプロテオミクス実験データ
**使用例:** 公開データ登録（レガシー）
**Pythonライブラリ:**
- `pyteomics.pride`
- カスタムXMLパーサー
**EDAアプローチ:**
- 実験メタデータ抽出
- 同定完全性
- スペクトルへのクロスリンク
- プロトコル情報
- 装置詳細

### .tsv / .csv (Proteomics)
**説明:** タブまたはカンマ区切りプロテオミクス結果
**典型的なデータ:** ペプチドまたはタンパク質定量テーブル
**使用例:** MaxQuant、Proteome Discoverer、Skyline出力
**Pythonライブラリ:**
- `pandas`: `pd.read_csv()` または `pd.read_table()`
**EDAアプローチ:**
- 同定数
- 定量値分布
- 欠損値パターン
- 強度ベース解析
- ラベルフリー定量評価
- アイソバリックタグ比解析
- 変動係数
- バッチ効果

### .msf - Thermo MSF Database
**説明:** Proteome Discoverer結果データベース
**典型的なデータ:** 検索結果を含むSQLiteデータベース
**使用例:** Thermo Proteome Discovererワークフロー
**Pythonライブラリ:**
- `sqlite3`: データベースアクセス
- カスタムMSFパーサー
**EDAアプローチ:**
- データベーススキーマ探索
- ペプチドおよびタンパク質テーブル
- スコア閾値
- 定量データ
- 処理ノード情報
- 信頼レベル

### .pdResult - Proteome Discoverer Result
**説明:** Proteome Discovererスタディ結果
**典型的なデータ:** 包括的な検索と定量
**使用例:** PDスタディエクスポート
**Pythonライブラリ:**
- 変換用ベンダーツール
- Python解析用にTSVへエクスポート
**EDAアプローチ:**
- スタディデザイン検証
- 結果フィルタリング基準
- 定量比較グループ
- 代入戦略

### .pep.xml - Peptide Summary
**説明:** コンパクトなペプチド同定形式
**典型的なデータ:** ペプチド配列、修飾、スコア
**使用例:** ダウンストリーム解析入力
**Pythonライブラリ:**
- `pyteomics`: XMLパース
**EDAアプローチ:**
- ユニークペプチドカウント
- PTMサイト局在
- 保持時間予測可能性
- チャージ状態嗜好

## 定量プロテオミクス

### .sky - Skyline Document
**説明:** Skylineターゲットプロテオミクスドキュメント
**典型的なデータ:** トランジションリスト、クロマトグラム、結果
**使用例:** ターゲットプロテオミクス（SRM/MRM/PRM）
**Pythonライブラリ:**
- `skyline`: Python API（限定的）
- 解析用にCSVへエクスポート
**EDAアプローチ:**
- トランジション選択検証
- クロマトグラフィックピーク品質
- 干渉検出
- 保持時間一貫性
- 検量線評価
- 反復相関
- LOD/LOQ決定

### .sky.zip - Zipped Skyline Document
**説明:** 外部ファイル付きSkylineドキュメント
**典型的なデータ:** 完全なSkyline解析
**使用例:** Skylineプロジェクト共有
**Pythonライブラリ:**
- `zipfile`: 処理用に展開
**EDAアプローチ:**
- ドキュメント構造
- 外部ファイル参照
- 結果エクスポートと解析

### .wiff - SCIEX WIFF Format
**説明:** 定量付きSCIEX装置データ
**典型的なデータ:** MRMトランジション付きLC-MS/MS
**使用例:** SCIEX QTRAP、TripleTOFデータ
**Pythonライブラリ:**
- ベンダーツール（限定的Pythonアクセス）
- mzMLへ変換
**EDAアプローチ:**
- MRMトランジション性能
- ドウェルタイム最適化
- サイクルタイム解析
- ピーク積分品質

### .raw (Thermo)
**説明:** Thermo生装置ファイル
**典型的なデータ:** Orbitrap、Q Exactiveからの完全MSデータ
**使用例:** ラベルフリーおよびTMT定量
**Pythonライブラリ:**
- `pymsfilereader`: Thermo RawFileReader
- `ThermoRawFileParser`: クロスプラットフォームCLI
**EDAアプローチ:**
- MS1およびMS2取得レート
- AGCターゲットとフィルタイム
- 分解能設定
- アイソレーションウィンドウ検証
- SPSイオン選択（TMT）
- 汚染評価

### .d (Agilent)
**説明:** Agilentデータディレクトリ
**典型的なデータ:** LC-MSおよびGC-MSデータ
**使用例:** Agilent装置ワークフロー
**Pythonライブラリ:**
- コミュニティパーサー
- mzMLへエクスポート
**EDAアプローチ:**
- メソッド一貫性
- キャリブレーションステータス
- シーケンス実行情報
- 保持時間安定性

## メタボロミクスおよび脂質omics

### .mzML (Metabolomics)
**説明:** メタボロミクス用標準MS形式
**典型的なデータ:** フルスキャンMS、ターゲットMS/MS
**使用例:** ターゲットなしおよびターゲットメタボロミクス
**Pythonライブラリ:**
- プロテオミクスmzMLツールと同様
**EDAアプローチ:**
- フィーチャー検出品質
- 質量精度評価
- 保持時間アライメント
- ブランク減算
- QC試料一貫性
- 同位体パターン検証
- アダクト形成解析
- インソースフラグメンテーションチェック

### .cdf / .netCDF - ANDI-MS
**説明:** MS用分析データ交換形式
**典型的なデータ:** GC-MS、LC-MSクロマトグラフィーデータ
**使用例:** メタボロミクス、GC-MSワークフロー
**Pythonライブラリ:**
- `netCDF4`: 低レベルアクセス
- `pyopenms`: CDFサポート
- `xcms` R統合経由
**EDAアプローチ:**
- TICおよび抽出イオンクロマトグラム
- 試料間ピーク検出
- 保持インデックス計算
- 質量スペクトルマッチング
- ライブラリ検索準備

### .msp - Mass Spectral Format (NIST)
**説明:** NISTスペクトルライブラリ形式
**典型的なデータ:** 参照質量スペクトル
**使用例:** 代謝物同定、ライブラリマッチング
**Pythonライブラリ:**
- `matchms`: スペクトルマッチング
- カスタムMSPパーサー
**EDAアプローチ:**
- ライブラリカバレッジ
- メタデータ完全性（InChI、SMILES）
- スペクトル品質メトリクス
- 衝突エネルギー標準化
- プレカーサータイプ注釈

### .mgf (Metabolomics)
**説明:** MS/MS用Mascot Generic Format
**典型的なデータ:** 代謝物同定用MS/MSスペクトル
**使用例:** スペクトルライブラリ検索
**Pythonライブラリ:**
- `matchms`: メタボロミクススペクトル解析
- `pyteomics.mgf`
**EDAアプローチ:**
- スペクトル品質フィルタリング
- プレカーサーアイソレーション純度
- フラグメントm/z精度
- ニュートラルロスパターン
- MS/MS完全性

### .nmrML - NMR Markup Language
**説明:** NMRメタボロミクス用標準XML形式
**典型的なデータ:** メタデータ付き1D/2D NMRスペクトル
**使用例:** NMRベースメタボロミクス
**Pythonライブラリ:**
- `nmrml2isa`: 形式変換
- カスタムXMLパーサー
**EDAアプローチ:**
- スペクトル品質メトリクス
- ビンning一貫性
- 参照化合物検証
- pHおよび温度効果
- 代謝物同定信頼度

### .json (Metabolomics)
**説明:** メタボロミクス結果用JSON形式
**典型的なデータ:** フィーチャーテーブル、注釈、メタデータ
**使用例:** GNPS、MetaboAnalyst、ウェブツール
**Pythonライブラリ:**
- `json`: 標準ライブラリ
- `pandas`: JSON正規化
**EDAアプローチ:**
- フィーチャー注釈カバレッジ
- GNPSクラスタリング結果
- 分子ネットワーキング統計
- アダクトおよびインソースフラグメントリンケージ
- 仮同定信頼度

### .txt (Metabolomics Tables)
**説明:** タブ区切りフィーチャーテーブル
**典型的なデータ:** 試料全体のm/z、RT、強度
**使用例:** MZmine、XCMS、MS-DIAL出力
**Pythonライブラリ:**
- `pandas`: テキストファイル読み込み
**EDAアプローチ:**
- フィーチャー数と品質
- 欠損値代入
- データ正規化評価
- バッチ補正検証
- QC用PCAとクラスタリング
- フォールドチェンジ計算
- 統計検定準備

### .featureXML - OpenMS Feature Format
**説明:** OpenMS検出フィーチャー
**典型的なデータ:** 品質スコア付きLC-MSフィーチャー
**使用例:** OpenMSワークフロー
**Pythonライブラリ:**
- `pyopenms`: FeatureXMLサポート
**EDAアプローチ:**
- フィーチャー検出パラメータ
- フィーチャーあたりの品質メトリクス
- 同位体パターンフィッティング
- チャージ状態割り当て
- FWHMと非対称性

### .consensusXML - OpenMS Consensus Features
**説明:** 試料間リンクフィーチャー
**典型的なデータ:** グループ情報付きアライメントフィーチャー
**使用例:** 複数試料LC-MS解析
**Pythonライブラリ:**
- `pyopenms`: ConsensusXML読み込み
**EDAアプローチ:**
- フィーチャー対応品質
- 保持時間アライメント
- 欠損値パターン
- 強度正規化必要性
- バッチ別フィーチャー一致

### .idXML - OpenMS Identification Format
**説明:** ペプチド/代謝物同定
**典型的なデータ:** スコア付きMS/MS同定
**使用例:** OpenMS IDワークフロー
**Pythonライブラリ:**
- `pyopenms`: IdXMLサポート
**EDAアプローチ:**
- 同定率
- スコア分布
- スペクトルマッチ品質
- 偽発見評価
- 注釈転送検証
