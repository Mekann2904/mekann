# バイオインフォマティクス・ゲノミクスファイル形式リファレンス

このリファレンスは、ゲノミクス、トランスクリプトミクス、配列解析、および関連するバイオインフォマティクスアプリケーションで使用されるファイル形式を網羅しています。

## 配列データ形式

### .fasta / .fa / .fna - FASTA Format
**説明:** ヌクレオチドまたはタンパク質配列のテキストベース形式
**典型的なデータ:** ヘッダー付きのDNA、RNA、またはタンパク質配列
**使用例:** 配列保存、BLAST検索、アライメント
**Pythonライブラリ:**
- `Biopython`: `SeqIO.parse('file.fasta', 'fasta')`
- `pyfaidx`: 高速インデックス付きFASTAアクセス
- `screed`: 高速配列パース
**EDAアプローチ:**
- 配列数と長さ分布
- GC含量解析
- N含量（不明確な塩基）
- 配列IDのパース
- 重複検出
- アセンブリ品質メトリクス（N50、L50）

### .fastq / .fq - FASTQ Format
**説明:** 塩基品質スコア付き配列データ
**典型的なデータ:** Phred品質スコア付きの生シーケンシングリード
**使用例:** NGSデータ、品質管理、リードマッピング
**Pythonライブラリ:**
- `Biopython`: `SeqIO.parse('file.fastq', 'fastq')`
- `pysam`: 高速FASTQ/BAM操作
- `HTSeq`: シーケンシングデータ解析
**EDAアプローチ:**
- リード数と長さ分布
- 品質スコア分布（塩基ごと、リードごと）
- GC含量とバイアス
- 重複率推定
- アダプター汚染検出
- k-mer頻度解析
- エンコーディング形式検証（Phred33/64）

### .sam - Sequence Alignment/Map
**説明:** アライメント用タブ区切りテキスト形式
**典型的なデータ:** マッピング品質付きアライメント済みシーケンシングリード
**使用例:** リードアライメント保存、変異検出
**Pythonライブラリ:**
- `pysam`: `pysam.AlignmentFile('file.sam', 'r')`
- `HTSeq`: `HTSeq.SAM_Reader('file.sam')`
**EDAアプローチ:**
- マッピング率と品質分布
- カバレッジ解析
- インサートサイズ分布（ペアエンド）
- アライメントフラグ分布
- CIGAR文字列パターン
- ミスマッチとインデル率
- 重複と補助アライメント数

### .bam - Binary Alignment/Map
**説明:** SAMの圧縮バイナリ版
**典型的なデータ:** 圧縮形式のアライメント済みリード
**使用例:** アライメントの効率的な保存と処理
**Pythonライブラリ:**
- `pysam`: インデックス付き完全BAMサポート
- `bamnostic`: Pure Python BAMリーダー
**EDAアプローチ:**
- SAMと同様、加えて:
- 圧縮率解析
- インデックスファイル（.bai）検証
- 染色体別統計
- ストランドバイアス検出
- リードグループ解析

### .cram - CRAM Format
**説明:** 高圧縮アライメント形式
**典型的なデータ:** 参照配列圧縮アライメント済みリード
**使用例:** 長期保存、スペース効率的アーカイブ
**Pythonライブラリ:**
- `pysam`: CRAMサポート（参照配列が必要）
- 参照ゲノムへのアクセスが必須
**EDAアプローチ:**
- BAMとの圧縮効率比較
- 参照依存性検証
- ロッシー/ロスレス圧縮評価
- 解凍パフォーマンス
- BAMと同様のアライメントメトリクス

### .bed - Browser Extensible Data
**説明:** ゲノムフィーチャー用タブ区切り形式
**典型的なデータ:** アノテーション付きゲノム間隔（chr、start、end）
**使用例:** ピークコーリング、変異アノテーション、ゲノムブラウジング
**Pythonライブラリ:**
- `pybedtools`: `pybedtools.BedTool('file.bed')`
- `pyranges`: `pyranges.read_bed('file.bed')`
- `pandas`: 簡易BED読み込み
**EDAアプローチ:**
- フィーチャー数とサイズ分布
- 染色体分布
- ストランドバイアス
- スコア分布（存在する場合）
- オーバーラップと近接性解析
- カバレッジ統計
- フィーチャー間ギャップ解析

### .bedGraph - BED with Graph Data
**説明:** 塩基ごとの信号値を持つBED形式
**典型的なデータ:** 連続値ゲノムデータ（カバレッジ、シグナル）
**使用例:** カバレッジトラック、ChIP-seqシグナル、メチル化
**Pythonライブラリ:**
- `pyBigWig`: bigWigへの変換可能
- `pybedtools`: BedGraph操作
**EDAアプローチ:**
- 信号分布統計
- ゲノムカバレッジ率
- 信号ダイナミクス（ピーク、バレー）
- 染色体別信号パターン
- 分位点解析
- ゼロカバレッジ領域

### .bigWig / .bw - Binary BigWig
**説明:** ゲノム全体の信号データ用インデックス付きバイナリ形式
**典型的なデータ:** 連続ゲノム信号（圧縮・インデックス付き）
**使用例:** 効率的なゲノムブラウザトラック、大規模データ
**Pythonライブラリ:**
- `pyBigWig`: `pyBigWig.open('file.bw')`
- `pybbi`: BigWig/BigBedインターフェース
**EDAアプローチ:**
- 信号統計抽出
- ズームレベル解析
- 領域信号抽出
- 効率的なゲノム全体サマリー
- 圧縮効率
- インデックス構造解析

### .bigBed / .bb - Binary BigBed
**説明:** インデックス付きバイナリBED形式
**典型的なデータ:** ゲノムフィーチャー（圧縮・インデックス付き）
**使用例:** 大規模フィーチャーセット、ゲノムブラウザ
**Pythonライブラリ:**
- `pybbi`: BigBed読み込み
- `pybigtools`: モダンなBigBedインターフェース
**EDAアプローチ:**
- フィーチャー密度解析
- 効率的な間隔クエリ
- ズームレベル検証
- インデックスパフォーマンスメトリクス
- フィーチャーサイズ統計

### .gff / .gff3 - General Feature Format
**説明:** ゲノムアノテーション用タブ区切り形式
**典型的なデータ:** 遺伝子モデル、転写産物、エクソン、調節エレメント
**使用例:** ゲノムアノテーション、遺伝子予測
**Pythonライブラリ:**
- `BCBio.GFF`: Biopython GFFモジュール
- `gffutils`: `gffutils.create_db('file.gff3')`
- `pyranges`: GFFサポート
**EDAアプローチ:**
- フィーチャータイプ分布（gene、exon、CDS等）
- 遺伝子構造検証
- ストランドバランス
- 階層関係検証
- CDSのフェーズ検証
- 属性完全性
- 遺伝子モデル統計（イントロン、遺伝子あたりのエクソン数）

### .gtf - Gene Transfer Format
**説明:** GFF2ベースの遺伝子アノテーション形式
**典型的なデータ:** 遺伝子と転写産物アノテーション
**使用例:** RNA-seq解析、遺伝子定量
**Pythonライブラリ:**
- `pyranges`: `pyranges.read_gtf('file.gtf')`
- `gffutils`: GTFデータベース作成
- `HTSeq`: カウント用GTF読み込み
**EDAアプローチ:**
- 転写産物アイソフォーム解析
- 遺伝子構造完全性
- エクソン数分布
- 転写産物長分布
- TSSとTES解析
- バイオタイプ分布
- オーバーラップ遺伝子検出

### .vcf - Variant Call Format
**説明:** 遺伝的変異用テキスト形式
**典型的なデータ:** アノテーション付きSNP、インデル、構造変異
**使用例:** 変異検出、集団遺伝学、GWAS
**Pythonライブラリ:**
- `pysam`: `pysam.VariantFile('file.vcf')`
- `cyvcf2`: 高速VCFパース
- `PyVCF`: 古いが包括的なライブラリ
**EDAアプローチ:**
- タイプ別変異数（SNP、インデル、SV）
- 品質スコア分布
- アレル頻度スペクトル
- 転換/非転換比
- ヘテロ接合率
- 欠損ジェノタイプ解析
- ハーディ・ワインベルグ平衡
- アノテーション完全性（アノテーション付きの場合）

### .bcf - Binary VCF
**説明:** 圧縮バイナリ変異形式
**典型的なデータ:** VCFと同じだがバイナリ
**使用例:** 効率的な変異保存と処理
**Pythonライブラリ:**
- `pysam`: 完全BCFサポート
- `cyvcf2`: 最適化BCF読み込み
**EDAアプローチ:**
- VCFと同様、加えて:
- 圧縮効率
- インデックス検証
- 読み込みパフォーマンスメトリクス

### .gvcf - Genomic VCF
**説明:** 参照信頼度ブロック付きVCF
**典型的なデータ:** 全位置（変異・非変異）
**使用例:** 共同ジェノタイピングワークフロー、GATK
**Pythonライブラリ:**
- `pysam`: GVCFサポート
- 標準VCFパーサー
**EDAアプローチ:**
- 参照ブロック解析
- カバレッジ均一性
- 変異密度
- ゲノム全体のジェノタイプ品質
- 参照信頼度分布

## RNA-Seqと発現データ

### .counts - Gene Count Matrix
**説明:** タブ区切り遺伝子発現カウント
**典型的なデータ:** サンプルごとのリードカウント付き遺伝子ID
**使用例:** RNA-seq定量、差異発現解析
**Pythonライブラリ:**
- `pandas`: `pd.read_csv('file.counts', sep='\t')`
- `scanpy`（シングルセル用）: `sc.read_csv()`
**EDAアプローチ:**
- ライブラリサイズ分布
- 検出率（サンプルあたりの遺伝子数）
- ゼロ過剰解析
- カウント分布（ログスケール）
- 外れ値サンプル検出
- レプリケート間相関
- サンプル関係のPCA

### .tpm / .fpkm - Normalized Expression
**説明:** 正規化遺伝子発現値
**典型的なデータ:** TPM（transcripts per million）またはFPKM値
**使用例:** サンプル間比較、可視化
**Pythonライブラリ:**
- `pandas`: 標準CSV読み込み
- `anndata`: 統合解析用
**EDAアプローチ:**
- 発現分布
- 高発現遺伝子の特定
- サンプルクラスタリング
- バッチ効果検出
- 変動係数解析
- ダイナミックレンジ評価

### .mtx - Matrix Market Format
**説明:** スパース行列形式（シングルセルで一般的）
**典型的なデータ:** スパースカウント行列（細胞 x 遺伝子）
**使用例:** シングルセルRNA-seq、大規模スパース行列
**Pythonライブラリ:**
- `scipy.io`: `scipy.io.mmread('file.mtx')`
- `scanpy`: `sc.read_mtx('file.mtx')`
**EDAアプローチ:**
- スパース性解析
- 細胞と遺伝子のフィルタリング閾値
- ダブレット検出メトリクス
- ミトコンドリア分画
- UMIカウント分布
- 細胞あたり遺伝子検出数

### .h5ad - Anndata Format
**説明:** HDF5ベースのアノテーション付きデータ行列
**典型的なデータ:** メタデータ付き発現行列（細胞、遺伝子）
**使用例:** Scanpyを用いたシングルセルRNA-seq解析
**Pythonライブラリ:**
- `scanpy`: `sc.read_h5ad('file.h5ad')`
- `anndata`: 直接AnnData操作
**EDAアプローチ:**
- 細胞数と遺伝子数
- メタデータ完全性
- レイヤー利用可能性（raw、normalized）
- 埋め込みの存在（PCA、UMAP）
- QCメトリクス分布
- バッチ情報
- 細胞タイプアノテーションカバレッジ

### .loom - Loom Format
**説明:** オミクスデータ用HDF5ベース形式
**典型的なデータ:** メタデータ付き発現行列
**使用例:** シングルセルデータ、RNA velocity解析
**Pythonライブラリ:**
- `loompy`: `loompy.connect('file.loom')`
- `scanpy`: loomファイルインポート可能
**EDAアプローチ:**
- レイヤー解析（spliced、unspliced）
- 行と列の属性探索
- グラフ接続性解析
- クラスター割り当て
- velocity固有メトリクス

### .rds - R Data Serialization
**説明:** Rオブジェクト保存（多くの場合Seuratオブジェクト）
**典型的なデータ:** R解析結果、特にシングルセル
**使用例:** R-Python間データ交換
**Pythonライブラリ:**
- `pyreadr`: `pyreadr.read_r('file.rds')`
- `rpy2`: 完全なR統合用
- AnnDataへの変換ツール
**EDAアプローチ:**
- オブジェクトタイプ特定
- データ構造探索
- メタデータ抽出
- 変換検証

## アライメントとアセンブリ形式

### .maf - Multiple Alignment Format
**説明:** 多重配列アライメント用テキスト形式
**典型的なデータ:** ゲノム全体または局所的多重アライメント
**使用例:** 比較ゲノミクス、保存性解析
**Pythonライブラリ:**
- `Biopython`: `AlignIO.parse('file.maf', 'maf')`
- `bx-python`: MAF固有ツール
**EDAアプローチ:**
- アライメントブロック統計
- 種カバレッジ
- ギャップ解析
- 保存性スコアリング
- アライメント品質メトリクス
- ブロック長分布

### .axt - Pairwise Alignment Format
**説明:** ペアワイズアライメント形式（UCSC）
**典型的なデータ:** ペアワイズゲノムアライメント
**使用例:** ゲノム比較、シンテニー解析
**Pythonライブラリ:**
- カスタムパーサー（単純な形式）
- `bx-python`: AXTサポート
**EDAアプローチ:**
- アライメントスコア分布
- 相同性パーセント
- シンテニックブロック特定
- ギャップサイズ解析
- カバレッジ統計

### .chain - Chain Alignment Format
**説明:** ゲノム座標マッピングチェーン
**典型的なデータ:** ゲノムビルド間の座標変換
**使用例:** Liftover、座標変換
**Pythonライブラリ:**
- `pyliftover`: chainファイル使用
- chain形式用カスタムパーサー
**EDAアプローチ:**
- チェーンスコア分布
- ソースゲノムのカバレッジ
- ギャップ解析
- 逆位検出
- マッピング品質評価

### .psl - Pattern Space Layout
**説明:** BLAT/BLASTアライメント形式
**典型的なデータ:** BLATからのアライメント結果
**使用例:** 転写産物マッピング、類似性検索
**Pythonライブラリ:**
- カスタムパーサー（タブ区切り）
- `pybedtools`: PSLを処理可能
**EDAアプローチ:**
- マッチパーセント分布
- ギャップ統計
- クエリカバレッジ
- 多重マッピング解析
- アライメント品質メトリクス

## ゲノムアセンブリとアノテーション

### .agp - Assembly Golden Path
**説明:** アセンブリ構造記述
**典型的なデータ:** スキャフォールド構成、ギャップ情報
**使用例:** ゲノムアセンブリ表現
**Pythonライブラリ:**
- カスタムパーサー（単純なタブ区切り）
- アセンブリ解析ツール
**EDAアプローチ:**
- スキャフォールド統計（N50、L50）
- ギャップタイプとサイズ分布
- コンポーネント長解析
- アセンブリ連続性メトリクス
- 未配置コンティグ解析

### .scaffolds / .contigs - Assembly Sequences
**説明:** アセンブリ配列（通常FASTA）
**典型的なデータ:** アセンブリ済みゲノム配列
**使用例:** ゲノムアセンブリ出力
**Pythonライブラリ:**
- FASTA形式と同じ
- アセンブリ固有ツール（QUAST）
**EDAアプローチ:**
- アセンブリ統計（N50、N90等）
- 長さ分布
- カバレッジ解析
- ギャップ（N）含量
- 重複評価
- BUSCO完全性（アノテーションがある場合）

### .2bit - Compressed Genome Format
**説明:** UCSCコンパクトゲノム形式
**典型的なデータ:** 参照ゲノム（高圧縮）
**使用例:** 効率的なゲノム保存とアクセス
**Pythonライブラリ:**
- `py2bit`: `py2bit.open('file.2bit')`
- `twobitreader`: 代替リーダー
**EDAアプローチ:**
- 圧縮効率
- ランダムアクセスパフォーマンス
- 配列抽出検証
- マスク領域解析
- N含量と分布

### .sizes - Chromosome Sizes
**説明:** 染色体長の単純形式
**典型的なデータ:** タブ区切りの染色体名とサイズ
**使用例:** ゲノムブラウザ、座標検証
**Pythonライブラリ:**
- pandasによる単純なファイル読み込み
- 多くのゲノムツールに組み込み
**EDAアプローチ:**
- ゲノムサイズ計算
- 染色体数
- サイズ分布
- 核型検証
- 参照との完全性チェック

## 系統解析と進化

### .nwk / .newick - Newick Tree Format
**説明:** 括弧表記ツリー表現
**典型的なデータ:** 分枝長付き系統樹
**使用例:** 進化解析、ツリー可視化
**Pythonライブラリ:**
- `Biopython`: `Phylo.read('file.nwk', 'newick')`
- `ete3`: `ete3.Tree('file.nwk')`
- `dendropy`: 系統解析
**EDAアプローチ:**
- ツリー構造解析（末端、内部ノード）
- 分枝長分布
- ツリーバランスメトリクス
- ウルトラメトリック性チェック
- ブートストラップサポート解析
- トポロジー検証

### .nexus - Nexus Format
**説明:** 系統データ用リッチ形式
**典型的なデータ:** アライメント、ツリー、形質行列
**使用例:** 系統解析ソフトウェア間交換
**Pythonライブラリ:**
- `Biopython`: Nexusサポート
- `dendropy`: 包括的Nexus処理
**EDAアプローチ:**
- データブロック解析
- 形質タイプ分布
- ツリーブロック検証
- タクサ一貫性
- コマンドブロックパース
- 形式準拠チェック

### .phylip - PHYLIP Format
**説明:** 配列アライメント形式（strict/relaxed）
**典型的なデータ:** 多重配列アライメント
**使用例:** 系統解析入力
**Pythonライブラリ:**
- `Biopython`: `AlignIO.read('file.phy', 'phylip')`
- `dendropy`: PHYLIPサポート
**EDAアプローチ:**
- アライメント次元
- 配列長均一性
- ギャップ位置解析
- 情報サイト計算
- 形式バリアント検出（strict vs relaxed）

### .paml - PAML Output
**説明:** PAML系統解析ソフトウェア出力
**典型的なデータ:** 進化モデル結果、dN/dS比
**使用例:** 分子進化解析
**Pythonライブラリ:**

- クロスリファレンス検証
- フィーチャーオーバーラップ解析

### .genbank / .gb / .gbk - GenBank Format
**説明:** NCBI配列アノテーション形式

- 起源生物抽出
- 参照と出版情報
- ローカスタグ一貫性

### .sff - Standard Flowgram Format
**説明:** 454/Rocheシーケンシングデータ形式

- キー配列検出
- アダプタートリミング検証

### .hdf5 (Genomics Specific)
**説明:** ゲノミクス用HDF5（10X、Hi-C等）

- 空間座標（該当する場合）

- TAD境界検出
