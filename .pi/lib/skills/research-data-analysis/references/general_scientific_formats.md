# 一般科学データ形式リファレンス

このリファレンスは、複数の分野で使用される汎用科学データ形式について説明します。

## 数値・配列データ

### .npy - NumPy Array
**説明:** NumPy配列のバイナリ形式
**典型的データ:** 任意のデータ型のN次元配列
**用途:** 数値データの高速I/O、中間結果の保存
**Pythonライブラリ:**
- `numpy`: `np.load('file.npy')`, `np.save()`
- Memory-mapped access: `np.load('file.npy', mmap_mode='r')`
**EDAアプローチ:**
- 配列の形状と次元数
- データ型と精度
- 統計サマリー（平均、標準偏差、最小値、最大値、パーセンタイル）
- 欠損値または無効値（NaN、inf）
- メモリフットプリント
- 値の分布とヒストグラム
- スパース性分析
- 相関構造（2次元配列の場合）

### .npz - Compressed NumPy Archive
**説明:** 複数のNumPy配列を1つのファイルに格納
**典型的データ:** 関連する配列のコレクション
**用途:** 複数配列のまとめて保存、圧縮ストレージ
**Pythonライブラリ:**
- `numpy`: `np.load('file.npz')` は辞書風オブジェクトを返す
- `np.savez()` または `np.savez_compressed()`
**EDAアプローチ:**
- 含まれる配列の一覧
- 個別の配列分析
- 配列間の関係
- ファイルサイズと圧縮率
- 命名規則
- データ整合性チェック

### .csv - Comma-Separated Values
**説明:** プレーンテキストの表形式データ
**典型的データ:** 実験測定値、結果テーブル
**用途:** 汎用データ交換、スプレッドシートエクスポート
**Pythonライブラリ:**
- `pandas`: `pd.read_csv('file.csv')`
- `csv`: 組み込みモジュール
- `polars`: 高性能CSV読み込み
- `numpy`: `np.loadtxt()` または `np.genfromtxt()`
**EDAアプローチ:**
- 行数と列数
- データ型推論
- 欠損値パターンと頻度
- 列統計（数値：平均、標準偏差；カテゴリカル：頻度）
- 外れ値検出
- 相関行列
- 重複行検出
- ヘッダーとインデックスの検証
- エンコーディング問題の検出

### .tsv / .tab - Tab-Separated Values
**説明:** タブ区切りの表形式データ
**典型的データ:** CSVと同様だがタブ区切り
**用途:** バイオインフォマティクス、テキスト処理出力
**Pythonライブラリ:**
- `pandas`: `pd.read_csv('file.tsv', sep='\t')`
**EDAアプローチ:**
- CSV形式と同様
- タブとスペースの検証
- 引用符処理

### .xlsx / .xls - Excel Spreadsheets
**説明:** Microsoft Excelバイナリ/XML形式
**典型的データ:** 書式設定や数式を含む表形式データ
**用途:** ラボノート、データ入力、レポート
**Pythonライブラリ:**
- `pandas`: `pd.read_excel('file.xlsx')`
- `openpyxl`: 完全なExcel操作
- `xlrd`: .xls読み込み（レガシー）
**EDAアプローチ:**
- シートの一覧と名前
- シートごとのデータ分析
- 数式の評価
- 結合セルの処理
- 非表示の行/列
- データ検証ルール
- 名前付き範囲
- 書式設定のみのセル検出

### .json - JavaScript Object Notation
**説明:** 階層的なテキストデータ形式
**典型的データ:** ネストされたデータ構造、メタデータ
**用途:** APIレスポンス、設定ファイル、結果データ
**Pythonライブラリ:**
- `json`: 組み込みモジュール
- `pandas`: `pd.read_json()`
- `ujson`: 高速JSON解析
**EDAアプローチ:**
- スキーマ推論
- ネスト深度
- キーと値の分布
- 配列長
- データ型の一貫性
- 欠損キー
- 重複検出
- サイズと複雑さのメトリクス

### .xml - Extensible Markup Language
**説明:** 階層的なマークアップ形式
**典型的データ:** メタデータ付きの構造化データ
**用途:** 標準ベースのデータ交換、API
**Pythonライブラリ:**
- `lxml`: `lxml.etree.parse()`
- `xml.etree.ElementTree`: 組み込みXML
- `xmltodict`: XMLを辞書に変換
**EDAアプローチ:**
- スキーマ/DTD検証
- 要素の階層と深度
- 名前空間処理
- 属性と要素コンテンツ
- CDATAセクション
- テキストコンテンツ抽出
- 兄弟要素と子要素の数

### .yaml / .yml - YAML
**説明:** 人間が読みやすいデータシリアライゼーション形式
**典型的データ:** 設定、メタデータ、パラメータ
**用途:** 実験設定、パイプライン定義
**Pythonライブラリ:**
- `yaml`: `yaml.safe_load()` または `yaml.load()`
- `ruamel.yaml`: YAML 1.2サポート
**EDAアプローチ:**
- 設定構造
- データ型処理
- リストと辞書の深度
- アンカーとエイリアスの使用
- 複数ドキュメントファイル
- コメントの保持
- スキーマに対する検証

### .toml - TOML Configuration
**説明:** 設定ファイル形式
**典型的データ:** 設定値、パラメータ
**用途:** Pythonパッケージ設定、アプリケーション設定
**Pythonライブラリ:**
- `tomli` / `tomllib`: TOML読み込み（tomllibはPython 3.11以降）
- `toml`: 読み書き対応
**EDAアプローチ:**
- セクション構造
- キーと値のペア
- データ型推論
- ネストされたテーブルの検証
- 必須フィールドとオプションフィールド

### .ini - INI Configuration
**説明:** シンプルな設定形式
**典型的データ:** アプリケーション設定
**用途:** レガシー設定、単純な設定
**Pythonライブラリ:**
- `configparser`: 組み込みINIパーサー
**EDAアプローチ:**
- セクション一覧
- キーと値の抽出
- 型変換
- コメント処理
- 大文字小文字の区別

## バイナリ・圧縮データ

### .hdf5 / .h5 - Hierarchical Data Format 5
**説明:** 大規模科学データセット用コンテナ
**典型的データ:** 多次元配列、メタデータ、グループ
**用途:** 大規模データセット、マルチモーダルデータ、並列I/O
**Pythonライブラリ:**
- `h5py`: `h5py.File('file.h5', 'r')`
- `pytables`: 高度なHDF5インターフェース
- `pandas`: HDFStore経由でHDF5保存
**EDAアプローチ:**
- グループとデータセットの階層
- データセットの形状とデータ型
- 属性とメタデータ
- 圧縮とチャンキング戦略
- メモリ効率的なサンプリング
- データセット間の関係
- ファイルサイズと効率
- アクセスパターンの最適化

### .zarr - Chunked Array Storage
**説明:** クラウド最適化されたチャンク配列ストレージ
**典型的データ:** 大規模N次元配列
**用途:** クラウドストレージ、並列計算、ストリーミング
**Pythonライブラリ:**
- `zarr`: `zarr.open('file.zarr')`
- `xarray`: Zarrバックエンドサポート
**EDAアプローチ:**
- 配列メタデータと次元
- チャンクサイズ最適化
- 圧縮コーデックと圧縮率
- 同期機構とストアタイプ
- マルチスケール階層
- 並列アクセスパフォーマンス
- 属性メタデータ

### .gz / .gzip - Gzip Compressed
**説明:** 圧縮データファイル
**典型的データ:** 圧縮されたテキストまたはバイナリ
**用途:** ストレージ/転送のための圧縮
**Pythonライブラリ:**
- `gzip`: 組み込みgzipモジュール
- `pandas`: 読み込み関数で自動gzip処理
**EDAアプローチ:**
- 圧縮率
- 元のファイルタイプ検出
- 解凍検証
- ヘッダー情報
- マルチメンバーアーカイブ

### .bz2 - Bzip2 Compressed
**説明:** Bzip2圧縮
**典型的データ:** 高圧縮ファイル
**用途:** gzipより高い圧縮率
**Pythonライブラリ:**
- `bz2`: 組み込みbz2モジュール
- pandasで自動処理
**EDAアプローチ:**
- 圧縮効率
- 解凍時間
- コンテンツ検証

### .zip - ZIP Archive
**説明:** 複数ファイルのアーカイブ
**典型的データ:** ファイルのコレクション
**用途:** ファイル配布、アーカイブ
**Pythonライブラリ:**
- `zipfile`: 組み込みZIPサポート
- `pandas`: ZIP圧縮CSV読み込み対応
**EDAアプローチ:**
- アーカイブメンバー一覧
- ファイルごとの圧縮方式
- 元のサイズと圧縮後サイズ
- ディレクトリ構造
- ファイルタイプ分布
- 解凍検証

### .tar / .tar.gz - TAR Archive
**説明:** Unixテープアーカイブ
**典型的データ:** 複数ファイルとディレクトリ
**用途:** ソフトウェア配布、バックアップ
**Pythonライブラリ:**
- `tarfile`: 組み込みTARサポート
**EDAアプローチ:**
- メンバーファイル一覧
- 圧縮（.tar.gz、.tar.bz2の場合）
- ディレクトリ構造
- パーミッション保持
- 解凍テスト

## 時系列・波形データ

### .wav - Waveform Audio
**説明:** オーディオ波形データ
**典型的データ:** 音響信号、オーディオ録音
**用途:** 音響分析、超音波、信号処理
**Pythonライブラリ:**
- `scipy.io.wavfile`: `scipy.io.wavfile.read()`
- `wave`: 組み込みモジュール
- `soundfile`: 拡張オーディオI/O
**EDAアプローチ:**
- サンプリングレートと継続時間
- ビット深度とチャンネル
- 振幅分布
- スペクトル分析（FFT）
- 信号対雑音比
- クリッピング検出
- 周波数成分

### .mat - MATLAB Data
**説明:** MATLABワークスペース変数
**典型的データ:** 配列、構造体、セル
**用途:** MATLAB-Python相互運用
**Pythonライブラリ:**
- `scipy.io`: `scipy.io.loadmat()`
- `h5py`: MATLAB v7.3ファイル（HDF5ベース）
- `mat73`: v7.3用の純粋Python
**EDAアプローチ:**
- 変数名と型
- 配列次元
- 構造体フィールド探索
- セル配列処理
- スパース行列検出
- MATLABバージョン互換性
- メタデータ抽出

### .edf - European Data Format
**説明:** 時系列データ（特に医療用）
**典型的データ:** EEG、生理信号
**用途:** 医療信号保存
**Pythonライブラリ:**
- `pyedflib`: EDF/EDF+読み書き
- `mne`: 神経生理学データ（EDF対応）
**EDAアプローチ:**
- 信号数と名前
- サンプリング周波数
- 信号範囲と単位
- 記録継続時間
- アノテーションイベント
- データ品質（飽和、ノイズ）
- 患者/スタディ情報

### .csv (Time Series)
**説明:** タイムスタンプ列を持つCSV
**典型的データ:** 時間インデックス付き測定値
**用途:** センサーデータ、モニタリング、実験
**Pythonライブラリ:**
- `pandas`: `pd.read_csv()` で `parse_dates` 使用
**EDAアプローチ:**
- 時間範囲と分解能
- サンプリング規則性
- 欠損時点
- トレンドと季節性
- 定常性テスト
- 自己相関
- 異常検出

## 地理空間・環境データ

### .shp - Shapefile
**説明:** 地理空間ベクトルデータ
**典型的データ:** 地理フィーチャ（ポイント、ライン、ポリゴン）
**用途:** GIS分析、空間データ
**Pythonライブラリ:**
- `geopandas`: `gpd.read_file('file.shp')`
- `fiona`: 低レベルシェープファイルアクセス
- `pyshp`: 純粋Pythonシェープファイルリーダー
**EDAアプローチ:**
- ジオメトリタイプと数
- 座標参照系
- 境界ボックス
- 属性テーブル分析
- ジオメトリ妥当性
- 空間分布
- マルチパートフィーチャ
- 関連ファイル（.shx、.dbf、.prj）

### .geojson - GeoJSON
**説明:** 地理データ用JSON形式
**典型的データ:** ジオメトリとプロパティを持つフィーチャ
**用途:** Webマッピング、空間分析
**Pythonライブラリ:**
- `geopandas`: ネイティブGeoJSONサポート
- `json`: JSONとして解析後に処理
**EDAアプローチ:**
- フィーチャ数とタイプ
- CRS仕様
- 境界ボックス計算
- プロパティスキーマ
- ジオメトリ複雑さ
- ネスト構造

### .tif / .tiff (Geospatial)
**説明:** 空間参照付きGeoTIFF
**典型的データ:** 衛星画像、DEM、ラスター
**用途:** リモートセンシング、地形分析
**Pythonライブラリ:**
- `rasterio`: `rasterio.open('file.tif')`
- `gdal`: Geospatial Data Abstraction Library
- `xarray` with `rioxarray`: N次元地理空間配列
**EDAアプローチ:**
- ラスター次元と解像度
- バンド数と説明
- 座標参照系
- ジオトランスフォームパラメータ
- NoData値処理
- ピクセル値分布
- ヒストグラム分析
- オーバービューとピラミッド

### .nc / .netcdf - Network Common Data Form
**説明:** 自己記述型配列ベースデータ
**典型的データ:** 気候、大気、海洋データ
**用途:** 科学データセット、モデル出力
**Pythonライブラリ:**
- `netCDF4`: `netCDF4.Dataset('file.nc')`
- `xarray`: `xr.open_dataset('file.nc')`
**EDAアプローチ:**
- 変数一覧
- 次元分析
- 時系列プロパティ
- 空間カバレッジ
- 属性メタデータ（CF規約）
- 座標系
- チャンキングと圧縮
- データ品質フラグ

### .grib / .grib2 - Gridded Binary
**説明:** 気象データ形式
**典型的データ:** 天気予報、気候データ
**用途:** 数値天気予報
**Pythonライブラリ:**
- `pygrib`: GRIBファイル読み込み
- `xarray` with `cfgrib`: GRIBからxarrayへ
**EDAアプローチ:**
- メッセージ一覧
- パラメータとレベルタイプ
- 空間グリッド仕様
- 時間カバレッジ
- アンサンブルメンバー
- 予報と解析
- データパッキングと精度

### .hdf4 - HDF4 Format
**説明:** 旧版HDF形式
**典型的データ:** NASA地球科学データ
**用途:** 衛星データ（MODIS等）
**Pythonライブラリ:**
- `pyhdf`: HDF4アクセス
- `gdal`: HDF4読み込み対応
**EDAアプローチ:**
- 科学データセット一覧
- Vdataと属性
- 次元スケール
- メタデータ抽出
- 品質フラグ
- HDF5またはNetCDFへの変換

- FITS規約準拠

### .asdf - Advanced Scientific Data Format

- エンコーディング検出

### .dat - Generic Data File
**説明:** バイナリまたはテキストデータ
**典型的データ:** 装置出力、カスタム形式
**用途:** 各種科学機器

**EDAアプローチ:**
- バイナリかテキストかの判定
- ヘッダー検出
- レコード構造推論
- エンディアン
- データ型パターン
- ドキュメントによる検証

**説明:** ソフトウェア/機器からのテキストログ
**典型的データ:** タイムスタンプ付きイベント、メッセージ
**用途:** トラブルシューティング、実験追跡
**Pythonライブラリ:**
- 組み込みファイル読み込み
- `pandas`: 構造化ログ解析
- 正規表現で解析
**EDAアプローチ:**
- ログレベル分布
- タイムスタンプ解析
- エラーと警告の頻度
- イベントシーケンス
- パターン認識
- 異常検出
- セッション境界
