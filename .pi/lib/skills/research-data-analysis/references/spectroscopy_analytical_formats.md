# 分光法・分析化学ファイル形式リファレンス

本リファレンスは、各種分光法技術および分析化学機器で使用されるファイル形式を網羅しています。

## NMR分光法

### .fid - NMR Free Induction Decay
**説明:** Bruker、Agilent、JEOLの生時間領域NMRデータ
**代表的データ:** 複素時間領域信号
**用途:** NMR分光法、構造解析
**Pythonライブラリ:**
- `nmrglue`: `nmrglue.bruker.read_fid('fid')` または `nmrglue.varian.read_fid('fid')`
- `nmrstarlib`: NMRデータ処理
**EDAアプローチ:**
- 時間領域信号の減衰
- サンプリングレートと取得時間
- データポイント数
- 信号対雑音比の推定
- ベースラインドリフトの評価
- デジタルフィルタ効果
- 取得パラメータの検証
- アポダイゼーション関数の選択

### .ft / .ft1 / .ft2 - NMR Frequency Domain
**説明:** フーリエ変換されたNMRスペクトル
**代表的データ:** 処理済み周波数領域データ
**用途:** NMR解析、ピーク積分
**Pythonライブラリ:**
- `nmrglue`: 周波数領域データの読み込み
- カスタム処理パイプライン
**EDAアプローチ:**
- ピーク検出と積分
- 化学シフト範囲
- ベースライン補正の品質
- 位相補正の評価
- 参照ピークの同定
- スペクトル分解能
- アーティファクト検出
- 多重度解析

### .1r / .2rr - Bruker NMR Processed Data
**説明:** Bruker処理済みスペクトル（実数部）
**代表的データ:** 1次元または2次元処理済みNMRスペクトル
**用途:** BrukerソフトウェアによるNMRデータ解析
**Pythonライブラリ:**
- `nmrglue`: Bruker形式サポート
**EDAアプローチ:**
- 処理パラメータの確認
- ウィンドウ関数の効果
- ゼロフィリングの評価
- 線形予測の検証
- スペクトルアーティファクト

### .dx - NMR JCAMP-DX
**説明:** NMR用JCAMP-DX形式
**代表的データ:** 標準化されたNMRスペクトル
**用途:** ソフトウェア間のデータ交換
**Pythonライブラリ:**
- `jcamp`: JCAMPリーダー
- `nmrglue`: JCAMPインポート対応
**EDAアプローチ:**
- 形式準拠性
- メタデータ完全性
- ピークテーブル検証
- 積分値
- 化合物同定情報

### .mnova - Mnova Format
**説明:** Mestrelab Research Mnova形式
**代表的データ:** 処理情報を含むNMRデータ
**用途:** Mnovaソフトウェアワークフロー
**Pythonライブラリ:**
- `nmrglue`: 限定的なMnovaサポート
- 標準形式への変換ツール
**EDAアプローチ:**
- マルチスペクトル処理
- 処理パイプラインの確認
- 定量データ
- 構造割り当て

## 質量分析

### .mzML - Mass Spectrometry Markup Language
**説明:** 標準XMLベースMS形式
**代表的データ:** MSスペクトル、クロマトグラム、メタデータ
**用途:** プロテオミクス、メタボロミクス、リピドミクス
**Pythonライブラリ:**
- `pymzml`: `pymzml.run.Reader('file.mzML')`
- `pyteomics.mzml`: `pyteomics.mzml.read('file.mzML')`
- `MSFileReader`: 各種ラッパー
**EDAアプローチ:**
- スキャン数とMSレベル分布
- 保持時間範囲とTIC
- m/z範囲と分解能
- プレカーサーイオン選択
- フラグメンテーションパターン
- 機器構成
- 品質管理指標
- データ完全性

### .mzXML - Mass Spectrometry XML
**説明:** レガシーXML MS形式
**代表的データ:** 質量スペクトルとクロマトグラム
**用途:** プロテオミクスワークフロー（旧式）
**Pythonライブラリ:**
- `pyteomics.mzxml`
- `pymzml`: mzXML読み込み対応
**EDAアプローチ:**
- mzMLと同様
- バージョン互換性
- 変換品質評価

### .mzData - mzData Format
**説明:** レガシーPSI MS形式
**代表的データ:** 質量分析データ
**用途:** レガシーデータアーカイブ
**Pythonライブラリ:**
- `pyteomics`: 限定的サポート
- mzMLへの変換を推奨
**EDAアプローチ:**
- 形式変換検証
- データ完全性
- メタデータ抽出

### .raw - Vendor Raw Files (Thermo, Agilent, Bruker)
**説明:** プロプライエタリ機器データ
**代表的データ:** 生質量スペクトルとメタデータ
**用途:** 機器からの直接出力
**Pythonライブラリ:**
- `pymsfilereader`: Thermo RAWファイル
- `ThermoRawFileParser`: CLIラッパー
- ベンダー固有API
**EDAアプローチ:**
- メソッドパラメータ抽出
- 機器パフォーマンス指標
- キャリブレーション状況
- スキャン機能解析
- MS/MS品質指標
- ダイナミック除外評価

### .d - Agilent Data Directory
**説明:** Agilent MSデータフォルダ
**代表的データ:** メソッド付きLC-MS、GC-MS
**用途:** Agilent MassHunterワークフロー
**Pythonライブラリ:**

**EDAアプローチ:**
- ディレクトリ構造検証
- メソッドパラメータ
- 検量線
- シーケンスメタデータ
- 信号品質指標

### .wiff - AB SCIEX Data
**説明:** AB SCIEX/SCIEX機器形式
**代表的データ:** 質量分析データ
**用途:** SCIEX機器ワークフロー
**Pythonライブラリ:**

**EDAアプローチ:**
- 実験タイプ同定
- スキャンプロパティ
- 定量データ
- マルチ実験構造

### .mgf - Mascot Generic Format
**説明:** MS/MS用ピークリスト形式
**代表的データ:** プレカーサーおよびフラグメント質量
**用途:** ペプチド同定、データベース検索
**Pythonライブラリ:**
- `pyteomics.mgf`: `pyteomics.mgf.read('file.mgf')`
- `pyopenms`: MGFサポート
**EDAアプローチ:**

- チャージ状態分布
- プレカーサーm/zおよび強度

- タイトルとメタデータ解析

### .pkl - Peak List (Binary)
**説明:** バイナリピークリスト形式
**代表的データ:** シリアライズされたMS/MSスペクトル
**用途:** ソフトウェア固有ストレージ
**Pythonライブラリ:**

**EDAアプローチ:**
- データ構造検査
- 標準形式への変換
- メタデータ保存

### .ms1 / .ms2 - MS1/MS2 Formats
**説明:** MSデータ用単純テキスト形式
**代表的データ:** MS1およびMS2スキャン
**用途:** データベース検索、プロテオミクス
**Pythonライブラリ:**

- 単純テキスト解析
**EDAアプローチ:**
- レベル別スキャン数
- 保持時間系列
- チャージ状態解析
- m/z範囲カバレッジ

### .pepXML - Peptide XML
**説明:** TPPペプチド同定形式
**代表的データ:** ペプチド-スペクトルマッチ
**用途:** プロテオミクス検索結果
**Pythonライブラリ:**
- `pyteomics.pepxml`
**EDAアプローチ:**
- 検索結果統計
- スコア分布
- 修飾解析
- FDR評価
- 酵素特異性

### .protXML - Protein XML
**説明:** TPPタンパク質推論形式
**代表的データ:** タンパク質同定
**用途:** プロテオミクスタンパク質レベル結果
**Pythonライブラリ:**
- `pyteomics.protxml`
**EDAアプローチ:**
- タンパク質グループ解析
- カバレッジ統計
- 信頼度スコアリング
- 節約解析

### .msp - NIST MS Search Format
**説明:** NISTスペクトルライブラリ形式
**代表的データ:** 参照質量スペクトル
**用途:** スペクトルライブラリ検索

## 赤外・ラマン分光法

### .spc - Galactic SPC
**説明:** Thermo Galactic分光法形式
**代表的データ:** IR、Raman、UV-Visスペクトル
**用途:** 各種分光法機器
**Pythonライブラリ:**
- `spc`: `spc.File('file.spc')`
- `specio`: マルチフォーマットリーダー
**EDAアプローチ:**
- 波数/波長範囲
- データポイント密度
- マルチスペクトル処理
- ベースライン特性
- ピーク同定
- 吸光度/透過率モード
- 機器情報

### .spa - Thermo Nicolet
**説明:** Thermo Fisher FTIR形式
**代表的データ:** FTIRスペクトル
**用途:** OMNICソフトウェアデータ
**Pythonライブラリ:**
- カスタムバイナリパーサー
- JCAMPまたはSPCへの変換
**EDAアプローチ:**
- 干渉図vsスペクトル
- バックグラウンドスペクトル検証
- 大気補正
- 分解能とスキャン数
- サンプル情報

### .0 - Bruker OPUS
**説明:** Bruker OPUS FTIR形式（番号付きファイル）
**代表的データ:** FTIRスペクトルとメタデータ
**用途:** Bruker FTIR機器
**Pythonライブラリ:**
- `brukeropusreader`: OPUS形式パーサー
- `specio`: OPUSサポート
**EDAアプローチ:**
- 複数ブロックタイプ（AB、ScSm等）
- サンプルおよび参照スペクトル
- 機器パラメータ
- 光路構成
- ビームスプリッターと検出器情報

### .dpt - Data Point Table
**説明:** 単純XYデータ形式
**代表的データ:** 汎用分光データ
**用途:** Renishaw Raman、一般的エクスポート
**Pythonライブラリ:**
- `pandas`: CSVライクな読み込み
- テキスト解析
**EDAアプローチ:**
- X軸タイプ（波長、波数、ラマンシフト）
- Y軸単位（強度、吸光度等）
- データポイント間隔
- ヘッダー情報
- マルチカラムデータ処理

### .wdf - Renishaw Raman
**説明:** Renishaw WiREデータ形式
**代表的データ:** Ramanスペクトルとマップ
**用途:** Renishaw Raman顕微鏡
**Pythonライブラリ:**
- `renishawWiRE`: WDFリーダー
- WDF形式用カスタムパーサー
**EDAアプローチ:**
- スペクトルvs マッピングデータ
- レーザー波長
- 積算と露光時間
- 空間座標（マッピング）
- Zスキャンデータ
- ベースラインと宇宙線除去

### .txt (Spectroscopy)
**説明:** 機器からの汎用テキストエクスポート
**代表的データ:** 波長/波数と強度
**用途:** 汎用データ交換
**Pythonライブラリ:**
- `pandas`: テキストファイル読み込み

**EDAアプローチ:**
- デリミタと形式検出
- ヘッダー解析
- 単位同定
- マルチスペクトル処理
- コメントからのメタデータ抽出

## X線・回折

- 結晶系と空間群

- 完全性と冗長性
- 構造検証

### .xy / .xye - Powder Diffraction

**代表的データ:** UPLC/HPLCクロマトグラム
**用途:** Waters機器データ
**Pythonライブラリ:**
- ベンダーツール（限定的Pythonアクセス）
- データベース抽出ツール
**EDAアプローチ:**
- 監査証跡情報
- 処理メソッド
- 化合物同定
- 定量結果
- システム適合性テスト

## その他の分析技術

### .dta - DSC/TGA Data
**説明:** 熱分析データ（TA Instruments）
**代表的データ:** 温度 vs 熱流または質量
**用途:** 示差走査熱量測定、熱重量測定
**Pythonライブラリ:**
- TA形式用カスタムパーサー
- エクスポートデータ用 `pandas`
**EDAアプローチ:**
- 転移温度同定
- エンタルピー計算
- 質量減少段階
- 昇温速度効果
- ベースライン決定
- 純度評価

### .run - ICP-MS/ICP-OES
**説明:** 元素分析データ
**代表的データ:** 元素濃度またはカウント
**用途:** 誘導結合プラズマMS/OES
**Pythonライブラリ:**
- ベンダー固有ツール
- カスタムパーサー
**EDAアプローチ:**
- 元素検出と定量
- 内標準パフォーマンス
- スパイク回収率
- 希釈係数補正
- 同位体比
- LOD/LOQ計算

### .exp - Electrochemistry Data
**説明:** 電気化学実験データ
**代表的データ:** 電位 vs 電流または電荷
**用途:** サイクリックボルタンメトリー、クロノアンペロメトリー
**Pythonライブラリ:**
- 機器固有カスタムパーサー（CHI、Gamry等）
- `galvani`: Biologic EC-Labファイル
**EDAアプローチ:**
- 酸化還元ピーク同定
- ピーク電位と電流
- 走査速度効果
- 電子移動速度論
- バックグラウンド減算
- キャパシタンス計算
