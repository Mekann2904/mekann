# セクションテンプレート集

SKILL.mdで使用できるセクションテンプレート集。コピー＆ペーストで使用可能。

## 概要セクション

### 基本概要

```markdown
## 概要

{スキル名}は{目的}を支援するスキル。

**主な機能:**
- 機能1の説明
- 機能2の説明
- 機能3の説明
```

### 詳細概要

```markdown
## 概要

{スキル名}は{目的}を支援するスキル。{詳細な説明}。

**主な機能:**
- **機能1**: 詳細説明
- **機能2**: 詳細説明
- **機能3**: 詳細説明

**対応範囲:**
- 対象1
- 対象2
- 対象3

**非対応範囲:**
- 非対象1
- 非対象2
```

## 使用タイミングセクション

### 基本パターン

```markdown
## 使用タイミング

以下の場合に使用:
- 使用シナリオ1
- 使用シナリオ2
- 使用シナリオ3
```

### 条件付きパターン

```markdown
## 使用タイミング

以下の場合に使用:
- 使用シナリオ1
- 使用シナリオ2

**特に以下の場合に推奨:**
- 推奨シナリオ1
- 推奨シナリオ2

**以下の場合は使用しない:**
- 非推奨シナリオ1
- 非推奨シナリオ2
```

```markdown

**{説明}**

2. ルール2
3. ルール3

### 違反時の影響

**重要**: このルールは必須（MANDATORY）です。例外なく遵守すること。
```

## スクリプトセクション

### 基本スクリプト

```markdown
## スクリプト

### scripts/{スクリプト名}.{拡張子}

```bash
# 使用方法
./scripts/{スクリプト名}.{拡張子} <引数>
```

**パラメータ:**
| 引数 | 型 | 必須 | 説明 |
|------|-----|------|------|
| arg1 | string | Yes | 説明 |
| arg2 | number | No | 説明 |
```

### 複数スクリプト

```markdown
## スクリプト

### scripts/setup.sh

初期セットアップ用スクリプト。

```bash
./scripts/setup.sh
```

### scripts/run.py

メイン実行スクリプト。

```bash
python scripts/run.py --input <file> --output <file>
```

### scripts/cleanup.sh

クリーンアップ用スクリプト。

```bash
./scripts/cleanup.sh
```
```

## 配置場所セクション

### プロジェクト固有の配置ルール

```markdown
## 配置場所

**推奨配置先:** `.pi/lib/skills/{skill-name}/`

| ディレクトリ | 用途 |
|--------------|------|
| `.pi/lib/skills/` | メインのスキル置き場 |
| `.pi/skills/` | 自動ロード回避用（通常は空） |

**設定:** `.pi/settings.json` に `{"skills": [".pi/lib/skills"]}` を追加
```
