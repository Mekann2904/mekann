## クイックスタート

5分でpi拡張機能コレクションを始めることができます。

- [クイックスタート](./01-quick-start.md) - 詳細情報

### ステップの概要

1. **piのインストール**
   ```bash
   npm install -g @mariozechner/pi-coding-agent
   ```

2. **fzfのインストール**
   ```bash
   brew install fzf  # macOS
   ```

3. **プロジェクトのセットアップ**
   ```bash
   pi install https://github.com/Mekann2904/mekann
   ```

4. **piの起動**
   ```bash
   pi
   ```
